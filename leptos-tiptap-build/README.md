# leptos-tiptap-build

Build dependencies for the [leptos-tiptap](https://crates.io/crates/leptos-tiptap) crate.

Check out the leptos-tiptap [repository](https://github.com/lpotthast/leptos-tiptap) for further instructions on how to
use this dependency.
